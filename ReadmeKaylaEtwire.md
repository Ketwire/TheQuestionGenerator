***To start:***
1.  **Open your terminal** and enter the folder.
2.  **Turn on your Python environment** based on your operating system:
    *   Linux/Mac: `source .venv/bin/activate`
    *   Windows: `.venv\Scripts\activate`
3.  **Install the magic tools** (only once):
    ```bash
    pip install fastapi uvicorn requests
    ```
4.  **Launch🚀 the app**:
    ```bash
    uvicorn main:app --reload --port 3003
    ```
5.  **Open your browser** and go to: `http://localhost:3003`


***All right. To play***🎮:
1. Type the **subject** required in the **subject** input box.
2. Type the **topic** you need in the **topic** input box.
3. Pick how many questions you want in the **number of questions** input box.
4. Click on the **generate questions** button.
5. 🪄Ta-da! And there you have it; your questions are ready!🎊
